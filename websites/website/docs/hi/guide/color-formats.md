# रंग प्रारूप

tsParticles `background`, `particles.paint` और प्लगइन सेटिंग्स जैसे विकल्पों में कई रंग प्रारूप स्वीकार करता है।

## सामान्य प्रारूप

```ts
color: "#60a5fa";
```

```ts
color: {
  value: {
    r: 96,
    g: 165,
    b: 250,
  },
}
```

```ts
color: {
  value: "hsl(220, 90%, 70%)",
}
```

## व्यावहारिक मार्गदर्शन

- दस्तावेज़ों और उदाहरणों में पठनीयता के लिए हेक्स को प्राथमिकता दें।
- समृद्ध यादृच्छिक दृश्यों के लिए रंगों की सारणी का उपयोग करें।
- जब टेक्स्ट के पीछे प्रभावों का उपयोग किया जाए तो कंट्रास्ट ऊंचा रखें।

# Plugin Option: Shapes

`shapes` provides individual shape packages that extend the built-in shape types.

## Available shapes

| Package | Shape name | Description |
| ------- | ---------- | ----------- |
| `shape-arrow` | `arrow` | Arrow shape |
| `shape-cards` | `cards` | Playing card shape |
| `shape-cog` | `cog` | Cog/gear shape |
| `shape-emoji` | `emoji` | Emoji character |
| `shape-heart` | `heart` | Heart shape |
| `shape-image` | `image` / `images` | External image or SVG |
| `shape-infinity` | `infinity` | Infinity symbol |
| `shape-matrix` | `matrix` | Matrix character rain |
| `shape-path` | `path` | Custom SVG path |
| `shape-polygon` | `polygon` | Regular polygon |
| `shape-ribbon` | `ribbon` | Ribbon shape |
| `shape-rounded-polygon` | `rounded-polygon` | Rounded-corner polygon |
| `shape-rounded-rect` | `rounded-rect` | Rounded rectangle |
| `shape-spiral` | `spiral` | Spiral shape |
| `shape-squircle` | `squircle` | Square-circle hybrid |
| `shape-star` | `star` | Star with N points |
| `shape-text` | `character` / `char` | Text character |

## Notes

- Select via `particles.shape.type`.
- Each shape may accept options under `particles.shape.options`.

## Pourquoi cette experience docs

Le site a ete repense autour de la question la plus frequente : **"Comment utiliser tsParticles aujourd'hui ?"**

- Partez d'un snippet fonctionnel en moins de 5 minutes.
- Choisissez le bon package sans explorer des dizaines de repositories.
- Trouvez rapidement options, configurations, presets et palettes.
- Lancez les effets lourds uniquement quand vous le decidez, avec des controles Start/Pause explicites.

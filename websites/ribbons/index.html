tsParticles Ribbons

Two quick examples: default ribbons and ribbons in a custom canvas.

CDN: <script src="https://cdn.jsdelivr.net/npm/@tsparticles/ribbons@__RIBBONS_VERSION__/tsparticles.ribbons.bundle.min.js"></script>

Ribbons Base

await ribbons({
  position: { x: random(0..100), y: 0 },
});

Ribbons Custom Canvas

const canvas = document.getElementById("customCanvas");
const localRibbons = await ribbons.create(canvas);

await localRibbons({
  position: { x: random(0..100), y: 0 },
});

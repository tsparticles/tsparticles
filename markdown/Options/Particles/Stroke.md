# Particles Stroke

Defines the outline drawn around particle shapes.

## Properties

| Key     | Type           | Example | Notes                                      |
| ------- | -------------- | ------- | ------------------------------------------ |
| `width` | `number`       | `2`     | Stroke width in pixels                     |
| `color` | `color object` |         | Stroke color, see {@link IAnimatableColor} |

## Quick example

```json
{
  "stroke": {
    "width": 2,
    "color": {
      "value": "#ffffff"
    }
  }
}
```

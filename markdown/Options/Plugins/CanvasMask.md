# CanvasMask Plugin Options

Options for the `canvasMask` plugin. This file documents common configuration accepted by the plugin's options classes.

Example configuration

```json
{
  "canvasMask": {
    "enable": true,
    "position": { "x": 0, "y": 0 },
    "size": { "width": 800, "height": 600 }
  }
}
```

Notes

- The exact shape of the `options` object depends on the plugin implementation and version. Keep this doc in sync with the plugin `Options` classes.

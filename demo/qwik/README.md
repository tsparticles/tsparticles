# Qwik Demo

Minimal Qwik demo for `@tsparticles/qwik`.

## Usage

From the repository root:

```bash
pnpm i
pnpm run build
cd demo/qwik
pnpm dev
```

Open the local URL shown by Vite (usually `http://localhost:5173`).

## Scripts

- `pnpm dev` starts the SSR dev server
- `pnpm start` starts dev server and opens the browser
- `pnpm build` builds the demo
- `pnpm preview` previews the production build

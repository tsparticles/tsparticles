# React Next.js Legacy Demo

Demo app for `@tsparticles/nextjs` using Next.js Pages Router.

## Scripts

```bash
pnpm run dev
pnpm run lint
pnpm run build
pnpm run start
```

## Notes

- `pages/_app.js` hosts `NextParticlesProvider` with the async init callback.
- `pages/index.js` renders `NextParticles`.
- `pages/api/hello.js` is a minimal API route sample.

# React Next.js Demo (App Router)

Demo app for `@tsparticles/react` using Next.js App Router.

## Scripts

```bash
pnpm run dev
pnpm run lint
pnpm run build
pnpm run start
```

## Notes

- `app/providers.tsx` mounts `ParticlesProvider` and runs async init once.
- `app/page.tsx` renders UI and particles.
- `pages/api/hello.ts` remains as API route sample.

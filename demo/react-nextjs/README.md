# React Next.js Demo (Pages Router)

Demo app for `@tsparticles/react` using Next.js Pages Router.

## Scripts

```bash
pnpm run dev
pnpm run lint
pnpm run build
pnpm run start
```

## Notes

- `pages/_app.js` hosts `ParticlesProvider` with the async init callback.
- `pages/index.js` renders the `Particles` component.
- `pages/api/hello.js` is a minimal API route sample.

# Ember Demo

Standalone Ember demo for `@tsparticles/ember`.

## Usage

From the repository root:

```bash
pnpm i
pnpm run build
cd demo/ember
pnpm start
```

Open `http://localhost:4200`.

## Scripts

- `pnpm start` starts Ember dev server
- `pnpm build` builds the demo app
- `pnpm build:ci` builds in production mode
- `pnpm test` runs demo tests

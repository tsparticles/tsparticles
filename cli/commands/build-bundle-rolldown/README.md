# tsParticles CLI - Build Bundle Rolldown

Internal command package used by `@tsparticles/cli-command-build` to run Rolldown-based bundle steps.

## Usage

From the CLI workspace root:

```bash
pnpm nx run @tsparticles/cli-command-build-bundle-rolldown:build
```

<template>

        <Particles

            background: {

                }
            },

                    onClick: {
                        enable: true,

                    }

            },

                color: {
                    value: '#ffffff'
                },

                    color: '#ffffff',
                    distance: 150,

                },

                    enable: true
                },

                    direction: 'none',
                    enable: true,
                    outMode: 'bounce',

                    speed: 6,
                    straight: false
                },
                number: {
                    density: {
                        enable: true,

                    value: 80

### Erros de TypeScript

Se o Typescript retornar algum erro ao importar/utilizar o plugin, tente adicionar o seguinte import antes do exemplo anterior:

```typescript
declare module "particles.vue3";
```

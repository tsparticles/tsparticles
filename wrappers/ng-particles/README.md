# Angular CLI  
  
![npm](https://img.shields.io/npm/v/ng-particles) ![npm](https://img.shields.io/npm/dm/ng-particles)  
  
## How to use it  
  
### Install  
  
```shell script  
npm install ng-particles  
```  
  
or  
  
```shell script  
yarn add ng-particles  
```  
  
### Usage  
  
*template.html*  
  
```html  
<ng-particles id="tsparticles" [options]="particlesOptions"></ng-particles>  
```  
  
*app.ts*  
  
```typescript

  fpsLimit: 60,
  interactivity: {
    detectsOn: "canvas",
```  
